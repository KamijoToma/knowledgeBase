# 账号申请与管理

## 注册

1. 进入[一网通办](https://ehall.ysu.edu.cn/ywtb-portal/ysu/index.html#/home)网站，搜索`个人邮箱办理`
  
    ![一网通办主页](/mail/申请1.png)

2. 点击`发起申请`

    ![代办事项列表](/mail/申请2.png)

3. 申请理由随便写(例如：申请教育优惠)、拟设邮箱随便写，不合法的随机分配

    ![申请表单](/mail/申请3.png)

4. 然后等待几天就好
